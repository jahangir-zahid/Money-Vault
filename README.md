Money-Vault is a simple and secure money collection web application designed for Father.
It helps track, manage, and record collected money in a clean and calm user interface.

This project is built using HTML, CSS, and JavaScript with smooth animations and an attractive modern design.

🚀 Features

👥 Access limited to only two users

💰 Add and manage money entries

📊 View total collected amount instantly

🎨 Calm and modern UI design

✨ Smooth animations and transitions

📱 Responsive layout

🔒 Simple login validation system

🖥️ Works locally without database

🛠️ Tech Stack

HTML5

CSS3 (Modern UI + Animations)

JavaScript (Logic & Data Handling)

🎯 Purpose of the Project

This project was created to:

Practice frontend development skills

Build a real-world mini finance management system

Strengthen JavaScript logic implementation

Create a useful application for personal family use
